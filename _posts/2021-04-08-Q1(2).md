[문제 상황]
가위, 바위, 보 게임을 만들어 봅시다. 가위바위
보 게임은 다음과 같이 진행됩니다. 
ex1)
입력(가위:s, 바위:r, 보:p): s
비겼다. 
ex2)
입력(가위:s, 바위:r, 보:p): k
잘못 입력됨
ex3)
입력(가위:s, 바위:r, 보:p): r
이겼다. 
ex4)
입력(가위:s, 바위:r, 보:p): p
졌다.

~~~python
import random
while True:
  try:
    ai = random.randrange(1,4)
    if ai == 1:
      ai = 's'
    elif ai == 2:
      ai = 'r'
    else:
      ai = 'p'
    user = input('입력(가위:s, 바위:r, 보:p): ')
    if not user == 's' and not user == 'r' and not user == 'p':
      print('잘못 입력됨')
    else:
      if ai == user:
        print('비겼다.')
      elif ai == 's' and user == 'r' or ai == 'r' and user == 'p' or ai == 'p' and user == 's':
          print('이겼다.')
      else:
          print('졌다.')
  except KeyboardInterrupt:
    break
~~~
